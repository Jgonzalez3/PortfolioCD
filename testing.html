Projects


FunRooms

Real-Time Multi-Game App & Message Room

Full-Stack multi-game team project developed with ASP.NET Core 2.1 server with SignalR Web Sockets for real-time application. Front-end developed primarily with vanilla JavaScript, along with jQuery and Bootstrap libraries. I was responsible for the development of the Tree Growing and Snail Race games.

Features: Social-Media 3rd Party Login, Real-Time Chat Rooms and Games, Camera Motion Detection Games.

  • Earth Defenders Game: 1 Player 2-D shooter game that utilizes game framework Phaser.io
  • Whiteboard Room: Multi-User game allowing multiple users to create hand drawn images together in real-time and message each other through chat box.
  • Tree Growing and Snail Racing Games: Both games are player vs. player utilizing hand gesture motion detection from each players camera. Developed with Gest.js library for motion detection and SignalR Web Sockets for real-time multiplayer functionality.

Wanderlust

E-Commerce Travel Products Website

  • RxJS: Products are queried in real-time with a library for Reactive programming using observables.
  • Facial Recognition: User can Login and Register with Facial Recognition utilizing Kairos API and WebRTC library.
